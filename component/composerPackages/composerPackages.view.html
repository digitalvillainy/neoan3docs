Neoan3 and Composer packages

Packagist is an aggregator of public PHP packages. Using a CLI tool known as Composer you can download and install any neoan3-app.

Pre-existing composer packages

As of this writing, Packagist has an lengthy list of neoan3-apps that were created by Stefan Roehrl, the mind behind Neoan3. Each of these apps have their source code available on Github.
